# 介绍

出于有趣和好玩起见，这是一本由知乎上收集整理的一些有趣的问题的答案集合，生成的“答案之书”，内容包括“**人生**”、“**健康**”、“**学习**”、“**逻辑**”、“**思维**”、“**时间**”等话题；
其中，问题主要来源于[知乎整理贴](https://zhuanlan.zhihu.com/p/25964484)及其他出处，所有答案均由ChatGPT的回答整合生成，并且保留了对应问题的链接在问题标题上，可用于与人工回答的答案进行对比；另一方面，“好”的问题本身就值得收藏和反复思考，由问题来启发“问题”，引导和链接其他问题，也是这个答案集合的题中之义；
此外，后续如有闲暇之余，计划将其重新整理为数据集，用来区分人工回答与机器回答的文本异同，从而对两者进行区分，获得更高质量的“知识”，而非是逻辑链生成的“胡言乱语”。
最后，谢邀，由GPT来回答。
我们一起来看：
